Bienvenue dans l'application TEOU

Visualiser votre position ou celle d'un contact sur une carte OpenStreetMap.

Chaque position est historisée dans l'espace de stockage du smartphone.

Utiliser l'application

Pour bénéficier de toutes les fonctionnalités, il faudra autoriser l'application à :

  • Contacts
  • Position
  • SMS
  • Stockage
  • Téléphone
